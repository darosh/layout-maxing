# layout-maxing

> A Max patch layout utility

[Live demo](https://darosh.github.io/layout-maxing)

## Development (Vite+ Monorepo)

- Check everything is ready:

```bash
vp run ready
```

- Run the tests:

```bash
vp run -r test
```

- Build the monorepo:

```bash
vp run -r build
```

- Run the development server:

```bash
vp run dev
```
